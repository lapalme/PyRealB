Query a flight database

Using an official RASA chat widget on the right, you can ask questions, in English, about flights, fares and airports. The answers are realized using pyrealb.

The information is available for flights between the following cities and airports: Atlanta, Denver, New York (JFK and La Guardia), Los Angeles, Chicago, Philadephia, Seattle and San Francisco. Only flights on United, American, US Airways, Delta and Virgin will be returned.

Make sure that the RASA server has been launched with rasa run command.
